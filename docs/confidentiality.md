# Confidentiality Note

This is a clean-room portfolio project. It uses synthetic reports, a generic
mini ESG dictionary, and newly written code.

It intentionally excludes:

- proprietary source code
- client or company datasets
- internal prompts or model configuration files
- cloud project IDs, bucket names, credentials, or job metadata
- generated JSON/JSONL/XLSX outputs from private work
- proprietary ESG scoring logic, materiality mappings, or validation thresholds

Before publishing, run a manual review and search for internal names, IDs,
local paths, credentials, generated outputs, and copied text.
